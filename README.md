# Healthy Recipes Finder

## Description

A front-end web project designed to help users discover healthy and easy-to-make recipes. The website provides a clean and modern interface that encourages users to explore nutritious meals without complexity.

It focuses on promoting healthy eating habits through simple navigation and visually appealing content.

## Overview

This project is a static multi-page website built using HTML5 and CSS3, without any backend or JavaScript functionality.

It is not just a landing page, but a small website that allows users to browse different sections such as home, about, and recipes in a structured and responsive layout.

## Website Pages

### Home (index.html)

The main page introduces the concept of healthy eating with:

* A hero section with a clear headline and call-to-action  
* A short description encouraging users to explore recipes  
* A clean and modern layout for first impressions  

### About (about.html)

Provides information about the purpose of the website, including:

* The idea behind healthy eating  
* The goal of simplifying meal choices  
* General information about the platform  

### Recipes (recipes.html)

Displays a collection of healthy recipes including:

* Recipe titles  
* Short descriptions  
* Organized layout for easy browsing  

## Features

* Clean and modern UI design  
* Easy navigation between pages  
* Call-to-action buttons (e.g., "Start Exploring", "Browse recipes")  
* Organized content structure  
* Fully responsive layout  

## Technologies Used

* HTML5  
* CSS3  
* CSS Flexbox / Grid  
* Responsive Design (mobile, tablet, desktop)  

## How to Run the Project

1. Download or clone the repository  
2. Open the project folder  
3. Run `index.html` in any modern web browser  
4. Use the navigation menu to explore the website  

## Design Notes

* Soft and natural color palette inspired by healthy food themes  
* Emphasis on simplicity and readability  
* Clean spacing and typography  
* Fully responsive without JavaScript  
* Designed for UI/UX practice and front-end development
